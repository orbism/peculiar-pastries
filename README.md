# Peculiar Pastries

Next.js website for a licensed New York cannabis cookie brand.

## Quick Start
```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Structure
```
/app              # Pages and layouts
/components       # Reusable UI components
/lib              # Utilities, data
/public
  /fonts          # Eames Century Modern
  /images         # Logos, brand assets
  /yum            # Product photography
```

## Notes
- No e-commerce - informational/educational only
- Products purchased at licensed NY dispensaries
- See CLAUDE.md for brand guidelines
